# 项目名称：工程型团队主页（Team Technical Portfolio Website）

------

# 1️⃣ 项目目标

构建一个**工程技术团队主页（不是个人作品集）**，用于系统性展示团队能力与项目经验。

核心要求：

- 展示以下技术方向能力：
  - 三维视觉 / 渲染引擎
  - SLAM / 无人机 / 三维重建
  - 大模型 / Agent / AI 工程化
  - 系统架构与高性能工程
- 必须包含当前所有项目（见下文）
- 支持后续**动态新增项目**
- 风格偏技术团队 / 工程团队，而非设计类作品集

------

# 2️⃣ 技术方案（强制要求）

使用：

- VitePress（静态网站生成器）
- Markdown 驱动内容
- 默认主题（避免复杂 UI）

约束：

- 必须支持 GitHub Pages 部署
- 不允许引入复杂前端框架（除非必要）
- 项目扩展不能依赖修改核心代码

------

# 3️⃣ 设计理念（必须体现）

整体风格应类似：

- 开源项目官网
- 技术团队介绍页
- 工程文档站

核心理念：

- 工程优先（Engineering-first）
- 系统化设计（System-oriented）
- 面向真实场景（Production-ready）
- AI 与工程结合（AI + System Integration）

------
